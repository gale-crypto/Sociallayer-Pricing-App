# Social Layer Pricing App

React + Vite app converted from the static pricing HTML. All copy is in English.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173). Use **Back to Home** to go to `/`, and **Pricing** or **View Pricing** to go to `/pricing`.

## Build

```bash
npm run build
npm run preview
```

## Stack

- **Vite** – build tool
- **React 18** – UI
- **React Router** – `/` (home) and `/pricing`
- **Tailwind CSS** – styling (same look as original HTML)
